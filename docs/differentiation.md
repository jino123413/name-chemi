## 고유 기능 상세

### 1. MagneticField SVG (자력선 시각화)
- 다른 앱에 없는 완전히 고유한 컴포넌트
- 5단계 끌림에 따라 자력선 곡선 수·밀도·색상·파티클 수 변화
- 두 이름이 N극/S극으로 표현

### 2. 2인 입력 → 리빌 → 결과 3단계 플로우
- 다른 운세 앱: 버튼 1번 → 즉시 결과
- 우리 케미: 이름 2개 입력 → 자석 리빌 애니메이션(2.8s) → 결과
- 리빌 중 끌림이(thinking) 마스코트가 분석 모션

### 3. 끌림이 마스코트 시스템
- 자석 요정 캐릭터 (LINE 스티커/카카오 프렌즈 플랫 스타일)
- 4종 × 3사이즈 = 12 PNG (main/happy/thinking/sad)
- 결과 레벨에 따라 마스코트 표정 자동 변경
- 고유 말투: "끌끌~", "자석처럼 딱!", "~거든!", "~잖아~"

### 4. 5속성 ECG 파동 (정보 덩어리화)
- 대화력/유머/감성/안정감/열정 5개 속성
- 다른 앱(3속성)보다 세밀한 분석
- 바 차트나 별 아이콘이 아닌 심전도 그래프 1개로 통합

### 5. 잘난체2 커스텀 폰트
- 다른 운세 앱: GmarketSans
- 우리 케미: 잘난체2 (로컬 TTF)
- 캐주얼하고 친근한 브랜드 느낌

## 절대 재사용하지 않은 요소

- ❌ 색/숫자/방향/시간 카테고리
- ❌ 4칸 그리드 (lucky-grid)
- ❌ 바 차트 (score-bar)
- ❌ S/A/B/C/D 등급
- ❌ 점수 (87점 등 숫자)
- ❌ 별/하트 아이콘 척도 (★★★★☆)
